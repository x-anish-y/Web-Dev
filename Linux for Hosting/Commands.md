# Linus Commands

## cd
### cd /home/user/documents: 
Changes the current directory to /home/user/documents. 
### cd ..: 
Moves to the parent directory of the current directory. 
### cd ~: 
Changes to the user's home directory. 
### cd: 
Without any arguments, cd will also take you to your home directory 
### cd -: 
Returns to the previous directory 

## touch
Creates new file
## touch .file_name
Creates a hidden file

## pwd
Shows the current directory path

## mkdir
Creates new directory

## cp (from directory path) (to directory path)
Copy one file and paste to (to directory path)

## mv (from directory path) (to directory path)
Move one file to (to directory path)

## ls
Give the list of all the folders and files of the current directory without hidden files
### ls -a
Give the list of all the folders and files of the current directory with hidden files
### ls /path/to/directory
Lists the files and directories within the specified path. 
### ls -l
Provides a detailed listing, including permissions, owner, group, size, modification date, and more. 
### ls -R
Performs a recursive listing, displaying the contents of subdirectories as well. 
### ls -t
 Sorts the output by the last modified time, with the most recently modified items listed first. 
### ls -S
Sorts the output by file size, with the largest files listed first. 
### ls -r
Reverses the order of the listing. 
### ls -F
Appends a character to each entry to indicate its type (e.g., '/' for directories, '*' for executables)
### ls -i
Displays the inode number of each file or directory.
